<WRAP professor>
Esta atividade visa explorar a formulação de conjecturas envolvendo a estrutura algébrica dos conjuntos numéricos. Neste momento, não se espera que os alunos justifiquem com rigor formal suas afirmações, mas sim que busquem ilustrar suas conjecturas a partir de exemplos. Recomenda-se que o professor discuta cada item a partir das soluções dos alunos, destacando as respostas corretas com base nos exemplos propostos pelos estudantes.
Com essas reflexões, visamos atingir não só reflexões a respeito de números racionais, mas também estimular a habilidade de argumentação em Matemática.
</WRAP>
</hidden>

Diga se as afirmações a seguir são verdadeiras ou falsas. Dê exemplos para ilustrar cada uma delas.
  - A soma de uma fração inteira com uma fração não inteira, é uma fração não inteira.
  - A diferença de uma fração inteira por uma fração não inteira é, necessariamente, uma fração não inteira.
  - A soma de uma fração não inteira por uma fração não inteira é, necessariamente, uma fração não inteira.
  - A diferença de uma fração não inteira por uma fração não inteira pode ser uma fração não inteira.
