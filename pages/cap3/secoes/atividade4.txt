==== Frações: a melhor escolha para dividir uma unidade ====
<WRAP professor>
Objetivos específicos:
Os alunos devem perceber que há boas escolhas para dividir uma unidade de modo a representar corretamente uma fração.

Recomendações:
A ideia é ainda continuar usando as faixas para reafirmar que só uma boa escolha da subdivisão a que a unidade será submetida permitirá uma representação pra esta parte ou para todo o elemento envolvido.
