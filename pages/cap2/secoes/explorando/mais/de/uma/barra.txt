O pai de Ana, Betriz e Clara trouxe duas barras de chocolate para serem repartidas entre elas.

{{ :cap2:secoes:explorando:mais:de:uma:chocolate-01.jpg?400 |}}

Ana propôs que cada barra fosse dividida em três partes iguais e que cada irmã ficasse com duas dessas partes.

{{ :cap2:secoes:explorando:mais:de:uma:chocolate-03.jpg?400 |}}


  - Na divisão de cada uma das barras de chocolate em três partes iguais, cada parte é que fração da barra?
  - Você concorda com essa divisão? Explique. 
  - Com essa divisão, as três irmãs receberiam a mesma quantidade de chocolate? 
  - Considerando-se uma barra inteira de chocolate, como você nomearia a quantidade de chocolate recebida por cada irmã usando a linguagem de frações?
  - Se Ana desse o que recebeu de chocolate para Beatriz, como você nomearia a quantidade de chocolate de Beatriz usando a linguagem de frações?  
  - Se Ana desse metade do que recebeu de chocolate para cada uma de suas irmãs, como você nomearia a quantidade de chocolate que elas ficariam usando a linguagem de frações?
