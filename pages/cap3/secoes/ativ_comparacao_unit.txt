Frações: comparando frações unitárias

<WRAP professor> 
Objetivos específicos:
Utilizar faixas para compreender a comparação entre frações unitárias diferentes. Na forma 1/b. Estimar a localização de cada fração unitária na reta

Recomendações:
Utilizar faixas com diferentes subdivisões para representar cada fração unitária. Pedir para que dada aluno construa uma reta numerada e localize nela as frações unitárias.

</WRAP>

Na figura abaixo temos partes pintadas de três faixas:
{{ :cap3:secoes:3_faixas.png?300 |}}

Para cada uma encontre as frações correspondentes e construa uma reta numerada e exiba uma localização aproximada para cada uma das frações.
